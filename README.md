# loglight

## 一、什么是 loglight ?

LightLog 是一个基于 C++20 开发的轻量级日志库，提供了灵活的日志记录功能。

## 二、目前实现了哪些

- [x] 日志输出到控制台
- [ ] 日志输出到文件


## 三、如何使用

1. 在 `linux` 终端中输入
``` bash
sh init.sh
```

2. 开始构建及用例运行
``` bash
sh build.sh
```
